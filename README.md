# Adaptive Evil Blocks

```
evil.block '@@test',

  render: -> ...
    @match '(max-width: 399px)', -> ...
    @match '(min-width: 400px)', -> ...

    @media '(max-width: 399px), match: -> ...
    @media '(min-width: 400px), match: -> ...

    @media '(max-width: 399px), match: -> ... , mismatch: -> ...
    @media '(min-width: 400px), match: -> ... , mismatch: -> ...

```
